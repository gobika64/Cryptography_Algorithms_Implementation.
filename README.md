# Implementation of Cryptographic Algorithms for Secure Data Communication

## Internship
Codec Technologies – Cyber Security Internship

## Objective
To implement and understand cryptographic algorithms used for securing data communication.

## Algorithms Implemented
- AES (Advanced Encryption Standard)
- RSA (Rivest–Shamir–Adleman)
- SHA-256 Hashing

## Tools & Technologies
- Python
- PyCryptodome Library

## Project Description
This project demonstrates the implementation of symmetric encryption, asymmetric encryption, and hashing techniques to ensure data confidentiality, integrity, and secure communication.

## How to Run
1. Install required library:
   pip install pycryptodome
2. Run the files:
   python aes.py
   python rsa.py
   python hashing.py

## Outcome
The project successfully encrypts, decrypts, and hashes data, demonstrating core cybersecurity concepts.
